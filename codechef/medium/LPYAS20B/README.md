## Problem

### Print Pattern

Write a program to output the square of * (stars) of size 4 (check the output below).

```
  
  
  
   

```

## Solution

**Language:** C++  
**Runtime:** N/A  
**Memory:** N/A  
**Submitted:** 2026-07-01T12:36:31.498Z  

```cpp
# cook your dish here

for i in range(4):  
    for j in range(4):
        print("*"", end="")
    print()
```
